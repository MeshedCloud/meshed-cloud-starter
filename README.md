# meshed-cloud-starter

项目通用starter管理

## 介绍
微服务快速开发流平台 [meshed-cloud](cloud.meshed.cn) 

**基础依赖管理工程**

推送制品库
```shell
mvn clean install org.apache.maven.plugins:maven-deploy-plugin:2.8:deploy -DskipTests
```
